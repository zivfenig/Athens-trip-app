# 🏛️ טיול יוון 2025

אפליקציה לתכנון טיול ביוון – מותאמת לסמארטפון, עובדת ב-GitHub Pages.

## 📁 מבנה הקבצים

```
├── index.html      ← ממשק המשתמש (UI בלבד)
├── app.js          ← כל הלוגיקה והנתונים
├── style.css       ← עיצוב
├── expenses.csv    ← תבנית לייצוא הוצאות
└── README.md
```

## 🚀 הפעלה דרך GitHub Pages

1. העלה את כל הקבצים לריפוזיטורי
2. לחץ **Settings** → **Pages**
3. תחת **Source** בחר **Branch: main** ולחץ **Save**
4. האפליקציה תהיה זמינה בכתובת: `https://[username].github.io/[repo-name]/`

## ⚙️ הגדרת Anthropic API Key

האפליקציה משתמשת ב-Anthropic API לחיפוש מידע וניתוח הוצאות.  
כדי שזה יעבוד, יש להוסיף את ה-API Key בקובץ `app.js` בכל קריאה ל-API:

```javascript
headers: {
  'Content-Type': 'application/json',
  'x-api-key': 'YOUR_API_KEY_HERE',
  'anthropic-version': '2023-06-01'
}
```

> **שים לב:** אל תעלה API Key לגיטהב! השתמש ב-GitHub Secrets או פרוס backend פשוט.
